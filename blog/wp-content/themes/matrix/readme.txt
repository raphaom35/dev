ABOUT THEME
=========== 
Matrix – Responsive Multi-Purpose Responsive HTML5 / CSS3 Theme for (Desktop, tablet, mobile phone…) – 100% Responsive Multi Purpose and Multi Pages Template. Matrix is Created with Twitter Bootstrap 3.3 Framework. Matrix is a great design idea for website of Creative Corporate, Corporation, Company Profile, Personal Portfolio, and more … Create Outstanding Website or Blog in Minutes! ... everything is possible with Matrix. Awesome Design, Unique Concepts, Scalable Responsive, Amazing flexibility, Dark and Light Scheme, Many Features & More …

This theme is compatible with Wordpress Version 4.3 and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).

Kirki toolkit (https://wordpress.org/plugins/kirki/) is used in to build customizer.

Supported browsers: Firefox, Opera, Chrome, Safari and IE10+ (Some css3 styles like shadows, rounder corners and 2D transform are not supported by IE8 and below).
